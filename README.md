# Git Intro Project

This repository contains my hands-on exercises for learning **Git and GitHub** as part of the AI Academy course at DLH.

---

## 📘 Objective

To understand and practice core version control concepts using Git, including working with local and remote repositories, branching, merging, and rollback strategies.

---

## 🧩 Topics Covered

* Initial Git setup and configuration
* Creating and managing repositories
* Staging and committing changes
* Using `.gitignore`
* Connecting to GitHub
* Push and pull operations
* Branch creation and switching
* Merging branches
* Conflict resolution
* Reverting and resetting changes
* Complete GitHub workflow (feature → PR → merge → rollback)

---

## ⚙️ Basic Commands Used

```bash id="a8k2lz"
git init
git status
git add .
git commit -m "message"
git push
git pull
git branch
git checkout
git merge
git revert
git reset --hard
```

---

## 🎯 Learning Outcome

By completing this module, I am able to:

* Track and manage code changes using Git
* Work with remote repositories on GitHub
* Use branching for feature development
* Resolve merge conflicts
* Safely undo changes when needed
